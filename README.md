# 📅 Wall Calendar App (React + Redux)

A modern, responsive **Wall Calendar UI** inspired by physical calendars, built using **React, Redux Toolkit, and Day.js**.  
This project focuses on clean UI, smooth UX, and real-world calendar behavior similar to Google Calendar.

---

## ✨ Features

### 📅 Calendar System
- Dynamic month view (previous, current, next dates)
- Full 7-column grid layout (always complete weeks)
- Month & year navigation (← →, +Y / -Y)

### 🎯 Date Selection
- Single date selection (default = today)
- Range selection (start → end)
- Smart selection handling (auto reset if needed)

### 🔄 Smart Navigation
- Clicking previous/next month dates auto-switches calendar
- Handles year transitions seamlessly

### 📝 Notes System
- Add notes per specific date
- Notes persist using localStorage
- Auto-load notes on date selection

### 🔹 Visual Indicators
- Dot marker for dates with notes
- Tooltip on hover to preview notes
- Highlighted selected date and range

### 🎨 UI/UX Enhancements
- Weekend styling (Sat/Sun in blue)
- Muted styling for non-current month dates
- Today highlight with dynamic behavior
- Clean shadows and modern typography

### 📱 Responsive Design
- Fully responsive (desktop + mobile)
- Proper spacing and layout adjustments for small screens

---

## 🛠 Tech Stack

- React.js
- Redux Toolkit
- Day.js
- CSS (custom styling)
